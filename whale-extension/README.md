# LinkScape 웨일 확장앱

브라우저 어디서든 링크·PDF·이미지를 LinkScape 캔버스로 바로 담는 확장.
빌드 도구 없이 순수 HTML/CSS/JS (Manifest V3) 로 되어 있어 이 폴더를 그대로 로드하면 된다.

## 설치 (네이버 웨일)

1. 주소창에 `whale://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장앱 설치** → 이 `whale-extension` 폴더 선택
4. 툴바의 L 아이콘 클릭 → LinkScape 계정으로 로그인

크롬에서도 동일하게 동작한다 (`chrome://extensions`).
설치 시 **"모든 웹사이트의 데이터를 읽고 변경할 수 있음"** 경고가 뜨는데,
모든 페이지에 드롭존을 심고 이미지를 대신 내려받기 위한 정상 고지다.

> 참고: 웨일의 `sidebar_action` 키는 구형(MV2) 확장 전용이라 MV3 의 `action` 과
> 같이 쓰면 "Only one of 'browser_action', ..." 오류로 로드가 거부된다. 그래서
> 이 확장은 툴바 팝업 + 페이지 드롭존으로 동작한다.

## 기능

| 동작 | 결과 |
| --- | --- |
| **페이지에서 드래그 시작** | 우하단에 드롭존 등장 — 링크·이미지·텍스트를 놓으면 저장. OS 에서 끌고 온 PDF·이미지 파일도 받는다 |
| **우클릭 → LinkScape에 링크 담기** | 그 링크가 카드로 (팝업 열 필요 없음) |
| **우클릭 → 이미지 담기** | 이미지 파일을 내려받아 업로드 (썸네일 포함, 실패 시 핫링크 카드 폴백) |
| **우클릭 → 메모로 담기** | 드래그 선택한 텍스트가 메모 카드로 |
| **우클릭 → 이 페이지 담기** | 현재 페이지가 링크 카드로 (탭 제목 사용) |
| **현재 탭 담기** 버튼 (팝업) | 지금 보고 있는 페이지가 링크 카드로 |
| 팝업에서 `Ctrl+V` | 클립보드의 URL → 링크 카드, 이미지 → 업로드, 일반 텍스트 → 메모 카드 |
| 팝업 URL 입력 후 Enter | 링크 카드 |
| 팝업에 PDF·이미지 드래그앤드롭 | 파일 업로드 + 카드 생성 (50MB 제한) |

- 드롭존 저장 결과는 페이지 위 **토스트**로, 우클릭 저장 결과는 확장 아이콘
  **배지**(파란 `✓` 성공, 빨간 `!` 실패 — 대부분 로그인 안 됨)로 표시된다.
- 새 카드는 보드에서 가장 최근 카드 근처(+32px 계단식)에 놓인다.
  보드를 새로고침하면 보인다.

## 동작 방식

- Supabase 에 REST 로 직접 붙는다 (`config.js` 의 anon key — 공개돼도 되는 키, 방어는 RLS).
- 세션은 `chrome.storage.local` 에 저장하고 만료 1분 전에 자동 갱신한다.
- OG 메타(썸네일·설명)는 수집하지 않는다 — 링크 카드는 호스트명 + 파비콘으로 시작한다.
- 페이지 이미지는 백그라운드 워커가 `<all_urls>` 권한으로 내려받아
  스토리지에 올린다 (드롭존은 CORS 때문에 직접 받을 수 없어 메시지로 위임).
- **PDF 썸네일·본문 텍스트는 웹사이트가 보드를 열 때 자동으로 채운다**
  (`src/components/board/usePdfBackfill.ts`). 확장은 업로드만 한다.
- 이미지 썸네일은 OffscreenCanvas 로 만들어 팝업·워커 어디서든 동작한다.
- 드롭존은 Shadow DOM 으로 페이지 CSS 와 격리된다.

## 파일 구조

```
manifest.json   MV3 매니페스트 (content_scripts + all_urls)
config.js       Supabase URL/anon key, 웹 주소
api.js          인증(패스워드/리프레시) · PostgREST · Storage 업로드
background.js   서비스 워커 — 우클릭 메뉴, 이미지 다운로드, 배지
dropzone.js     콘텐츠 스크립트 — 드래그 감지, 드롭 타겟, 토스트
popup.html/css  UI — 웹사이트 디자인 토큰(DESIGN-apple.md)을 그대로 복제
popup.js        붙여넣기/드롭/현재 탭/작업 목록 로직
icons/          생성된 아이콘 (Action Blue + L)
```

일반 탭에서 `popup.html` 을 직접 열어도 동작한다 (chrome.* API 없으면
localStorage 폴백) — 개발할 때 편하다.
